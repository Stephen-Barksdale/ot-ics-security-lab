# OT/ICS Security Lab

## Purpose

This repository demonstrates operational technology cybersecurity concepts, including OT asset visibility, network segmentation, vulnerability remediation, risk assessment, and incident response planning.

This lab is aligned to roles involving OT/ICS cybersecurity, SCADA security, BAS/BMS environments, Army OT/IT systems, industrial network security, and mission-critical infrastructure.

## Skills Demonstrated

- OT asset inventory development
- OT/IT risk assessment
- BACnet and Modbus security considerations
- Network segmentation concepts
- Vulnerability remediation planning
- POA&M-style tracking
- OT incident response planning
- Operational availability awareness

## Tools and Concepts

- BACnet
- Modbus
- SCADA concepts
- BAS/BMS systems
- Gateways
- Controllers
- Telemetry
- Firewalls
- VLANs
- ACLs
- Nessus/OpenVAS concepts
- NIST 800-82
- NIST 800-53
- RMF
- POA&M
